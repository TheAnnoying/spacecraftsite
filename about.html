Space Craft is a Minecraft server created by TheAnnoying. The idea is quite simple, and inspired by a Minehut server called "WWWings".

  • • You're contained inside a box in space.
  • • You can build inside the box using the blocks you buy from the shop. Those blocks could also be used for decorative building and protection.
  • • If you fall too far down, you'll die.
  • • You earn coins by killing other players, and that could be achieved using the kits or explosives from the shop.
  • • You can use gadgets such as the jetpack to fly around!
  • • Cosmetics/balloons will be added soon!
